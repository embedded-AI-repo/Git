# pull保持更新
## pull的示例场景
Melody正在和她的团队一起进行一个黑客马拉松项目。她在一个叫做 的分支工作，负责为她正在开发的应用添加一个功能。feature

她遇到了一个队友艾米丽之前已经解决过的问题，于是请求艾米丽帮忙。艾米丽切换到梅洛迪的分支，使用，并识别出窃听器。git checkout feature

Emily修改了分支上的几行代码并推送她的更改。Melody 现在希望这些更改反映在她电脑上的仓库版本中，于是她使用了该命令。Melody看到Emily修改的代码行数，继续开发该功能。featuregit pull
